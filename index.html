Fase de Grupos

🥇3 puntos — Marcador exacto
✅1 punto — Acertaste el ganador o el empate

Eliminatorias

🥇3 puntos — Marcador exacto en tiempo regular/extra
✅1 punto — Acertaste el ganador o el empate en tiempo regular/extra
⚽+0.5 puntos — Por cada equipo cuyos goles acertaste (solo si no hay exacto)
🥅+2 puntos bonus — Predijiste que iba a penaltis y acertaste el equipo ganador de la tanda
❌-1 punto — Predijiste penaltis pero el partido se resolvió en tiempo reglamentario o extra

Ejemplos

Predijiste 2-1, resultado 2-1 → 3 pts

Predijiste 3-1, resultado 2-1 → 1 + 0.5 = 1.5 pts

Predijiste 2-0, resultado 2-1 → 1 + 0.5 = 1.5 pts

Predijiste 0-1, resultado 2-1 → 0 + 0.5 = 0.5 pts

Predijiste 2-1 + penaltis + Local, termina 1-1, Local gana penaltis → 0 + 0.5 + 2 = 2.5 pts

Predijiste 1-0 + penaltis, termina 2-1 sin penaltis → 1 - 1 = 0 pts

Predijiste 0-0 + penaltis, termina 2-1 sin penaltis → 0 - 1 = -1 pt

⚠️ Importante

• Las predicciones se cierran al inicio de cada partido

• Solo puedes predecir partidos habilitados por el organizador

• En eliminatorias el resultado de penaltis NO afecta el marcador registrado
